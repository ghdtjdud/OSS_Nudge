# Nudge

무기력하거나 일상 루틴을 시작하기 어려운 사용자가 작은 생활 행동을 수행할 수 있도록 돕는 AI 기반 생활 루틴 지원 서비스입니다.
사용자의 음성 입력을 바탕으로 현재 상태를 파악하고, 물 마시기·양치하기·식사하기·복약하기와 같은 작은 미션을 제안합니다.
사용자는 미션 수행 후 카메라 인증을 통해 완료 여부를 확인하고, 대시보드에서 월별 수행 기록을 시각적으로 확인할 수 있습니다.

## 개발 배경

현대인은 바쁜 일정, 스트레스, 무기력감 등으로 인해 기본적인 생활 루틴을 놓치는 경우가 많습니다. 특히 물 마시기, 양치하기, 식사하기, 약 복용하기와 같은 작은 행동은 건강과 일상 유지에 중요하지만, 심리적으로 지치거나 생활 리듬이 무너진 상황에서는 시작 자체가 어려울 수 있습니다.
기존의 루틴 관리 서비스는 사용자가 직접 계획을 세우고 완료 여부를 수동으로 기록해야 하는 경우가 많습니다. 그러나 무기력한 상태의 사용자에게는 계획 수립과 기록 과정마저 부담이 될 수 있습니다. 이에 따라 사용자의 현재 상태를 자연스럽게 파악하고, 바로 실천 가능한 작은 행동을 제안하며, 수행 결과를 시각적으로 보여주는 서비스가 필요하다고 판단하여, 사용자가 부담 없이 일상 행동을 시작할 수 있도록 돕는 것을 목표로 개발되었습니다.

## 개발 목적 및 기대효과

[개발 목적]
Nudge의 목적은 사용자의 일상 회복을 돕는 작은 행동 기반 미션 서비스를 제공하는 것입니다. 사용자는 채팅 또는 음성 입력을 통해 자신의 상태를 표현하고, 서비스는 해당 상황에 맞는 미션을 추천합니다. 이후 사용자는 제안된 미션을 수행하고, 카메라 인증을 통해 실제 수행 여부를 확인할 수 있습니다.
본 프로젝트는 단순한 할 일 관리가 아니라, 사용자의 현재 상태를 기반으로 적절한 생활 행동을 제안하고 수행 경험을 긍정적으로 강화하는 데 중점을 두었습니다.

[기대 효과]
사용자가 작은 행동부터 시작하며 일상 루틴을 회복할 수 있습니다.
채팅과 음성 입력을 통해 사용자가 부담 없이 자신의 상태를 표현할 수 있습니다.
AI 기반 미션 추천을 통해 사용자의 상황에 맞는 행동을 제안할 수 있습니다.
카메라 기반 인증을 통해 미션 수행 여부를 보다 직관적으로 확인할 수 있습니다.
월간 대시보드와 스티커 기록을 통해 사용자가 자신의 성취를 시각적으로 확인할 수 있습니다.
복약, 식사, 수분 섭취, 양치 등 기본 생활 행동을 꾸준히 관리하는 데 도움을 줄 수 있습니다.

## 주요 기능

[회원가입 및 로그인]
사용자는 회원가입 후 로그인하여 서비스를 이용할 수 있습니다. 로그인 시 발급된 JWT 토큰을 기반으로 인증이 필요한 API에 접근합니다.

[사용자 상태 초기 설정]
서비스 이용 전 사용자의 생활 상태를 파악하기 위한 초기 상태 설정을 진행합니다.
수면 시간, 식사 규칙성, 복약 여부, 활동 시작의 어려움 등을 입력받아 이후 미션 추천 및 사용자 관리에 활용합니다.

[AI 채팅 및 음성 인식]
사용자는 텍스트 채팅 또는 음성 입력을 통해 자신의 현재 상태를 표현할 수 있습니다.
음성 입력의 경우 STT를 통해 텍스트로 변환한 뒤, AI가 사용자의 상태를 분석하여 적절한 응답과 미션을 생성합니다.
예를 들어 사용자가 “오늘 물을 거의 못 마셨어”라고 입력하면 물 마시기 미션이 추천될 수 있고, “하루 종일 누워있어서 양치도 못 했어”라고 입력하면 양치 미션이 추천될 수 있습니다.

[미션 카드]
AI가 미션을 추천하면 사용자는 미션 카드 화면에서 수행할 미션을 확인할 수 있습니다.
미션 카드는 사용자가 해야 할 행동을 간단하고 명확하게 안내하며, 미션 시작 후 인증 화면으로 이동합니다.

[카메라 기반 미션 인증]
사용자는 미션 수행 후 카메라를 통해 인증을 진행합니다.
프론트엔드는 일정 간격으로 카메라 프레임을 서버에 전송하고, 백엔드는 YOLOv12 기반 객체 탐지 모델을 활용하여 미션 수행 여부를 판단합니다.
인증 성공 시 성공 피드백 화면으로 이동하고, 일정 시간 내 인증에 실패하면 실패 피드백 화면으로 이동합니다.

[피드백 화면]
미션 인증 결과에 따라 성공 또는 실패 피드백을 제공합니다.
성공 시 사용자가 수행한 행동을 긍정적으로 강화하는 메시지를 보여주고, 대시보드로 이동할 수 있습니다. 실패 시 다시 미션을 시도할 수 있도록 안내합니다.

[대시보드 캘린더]
대시보드에서는 사용자의 월별 미션 완료 기록을 캘린더 형태로 확인할 수 있습니다.
각 날짜에는 완료한 서로 다른 미션 종류의 개수에 따라 스티커가 표시됩니다.
미션 완료 기록은 미션 배정일이 아니라 실제 인증 완료 시간인 completed_at을 기준으로 집계됩니다. 이를 통해 사용자는 자신이 실제로 행동을 수행한 날짜를 기준으로 기록을 확인할 수 있습니다.

[마이페이지]
마이페이지에서는 사용자의 기본 회원정보와 계정 정보를 관리할 수 있습니다.

## 환경설정 및 실행 방법

[프로젝트 클론]
```bash
git clone <repository-url>
cd Nudge
```

[백엔드 실행]
가상환경을 생성하고 활성화합니다.
```bash
python3.11 -m venv .venv
source .venv/bin/activate
```

필요한 패키지를 설치합니다.
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

카메라 기반 미션 인증 기능은 YOLOv12 기반 객체 탐지 모델을 사용하므로, 다음 명령으로 YOLOv12 전용 패키지를 설치해야 합니다.
```bash
pip uninstall -y ultralytics
pip install -q git+https://github.com/sunsmarterjie/yolov12.git
```

.env 파일을 프로젝트 루트에 생성하고 필요한 환경변수를 설정합니다.
```
DATABASE_URL=<database-url>
JWT_SECRET_KEY=<jwt-secret-key>
GEMINI_API_KEY=<gemini-api-key>
DEV_CHAT_ENABLED=true
```

백엔드 서버를 실행합니다.
```bash
set -a
source .env
set +a
uvicorn backend.app.main:app --reload
```

서버 실행 후 Swagger 문서는 아래 주소에서 확인할 수 있습니다.
http://localhost:8000/docs

[프론트엔드 실행]
프론트엔드 의존성을 설치합니다.
```bash
npm install
```

프론트엔드 개발 서버를 실행합니다.
```bash
npm run dev
```

실행 후 브라우저에서 아래 주소로 접속합니다.
http://localhost:5173

포트가 이미 사용 중인 경우 Vite가 다른 포트를 사용할 수 있습니다.

## 프로젝트 의의

Nudge는 단순한 루틴 체크리스트가 아니라, 사용자의 현재 상태를 기반으로 작은 행동을 제안하고 수행을 돕는 서비스입니다. 특히 무기력하거나 일상 루틴이 무너진 상황에서도 사용자가 부담 없이 행동을 시작할 수 있도록 설계했습니다.
또한 AI 채팅, 음성 인식, 객체 탐지, 캘린더 시각화 기능을 하나의 사용자 흐름으로 연결하여, 사용자의 입력부터 행동 추천, 인증, 기록 확인까지 이어지는 통합적인 경험을 제공한다는 점에서 의의가 있습니다.
